## CPCG (Cross Platform Component Generator)

> Create beautiful and optimized components using GUI or from `JSON` for your next project using any framework - `Vue.js` or `React.js` or similar.

## Tech Stack

1. Vue.js
2. Tailwind CSS
3. Shadcn
